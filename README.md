# Sales Data Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents a Sales Dashboard created using Power BI. It analyzes yearly sales data to identify trends and insights.

## 📁 Data Sources
- Customer Table
- Orders Table
- Product Table

## 🔗 Data Modeling
- Created relationships between tables
- Performed data cleaning and transformation

## 📈 Key Features
- Interactive dashboard
- Two report pages (Page 1 & Page 2)
- Sales insights and performance tracking

## 🧮 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)

## 📂 Files Included
- Sales Dashboard (PDF) – for quick preview
- Power BI file (.pbix) – for full project analysis
